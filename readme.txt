
スタンプ帳　version 1.4

これはゴーストのスタンプを収集するSSP専用プラグインです。
あまり古すぎるSSPには対応していません。


最初はまっさらなスタンプ帳があるだけです。
対応しているゴーストから押してもらってください。
スタンプを押してもらえるゴースト一覧はこちらです。
 > http://www10.atwiki.jp/postic/pages/13.html


スタンプはゴースト一体につき一つまでです。
貰える条件はそのゴーストによって違います。
出会ってすぐにあっさり押して貰えるかもしれませんし、なかなか貰えないかもしれません。

押してもらったスタンプは
右クリックメニュー＞プラグイン＞スタンプ帳
で確認できます。
小さいバルーンでは上手く表示されません（対応法考え中）

よくわからない方はこちらの解説が詳しいのでぜひ。
http://d.hatena.ne.jp/hinoharu/20080507#1210173143




ほか、細かいこと。

■スタンプを削除する
スタンプは追加、上書きされるだけで削除はできません。
どうしても一旦初期化させたい場合はゴーストが起動していない状態でssp.exeと同じ位置に存在する「plugin」内「stampcollection」フォルダを開き、「YAYA_variable.cfg.ays」を削除すれば元のスタンプが1つもない状態に戻ります。

■スタンプをバックアップする、他のSSPに移動する
ssp.exeと同じ位置に存在する「plugin」内「stampcollection」フォルダを、移動先SSPの同じ位置にコピーすると移動できます。
（スタンプのデータは「YAYA_variable.cfg.ays」と「icon」フォルダの中身全てです）
バックアップする場合はこのフォルダをどこか別な場所にコピーして保管しておいてください。

■スタンプをネットワーク更新する
エクスプローラーを開き、「種類」を「プラグイン」にし、スタンプ帳項目右クリックからできます。
（それ以外の方法もあるのかもしれませんがよくわかりません）

■スタンプ帳プラグインの扱いについて
再配布・ゴーストに同梱配布可、改変配布可です。
（2013/12/28　改変配布不可より可に変更））
